# DinosaurGame-chrome
## ***Requirements***  
- Python 3.5 or above
- opencv 4.0 or above
### Create Anaconda env:
```bash
conda create --name "name of env"
```
### Install Requirements by conda or pip After Activate env:
```bash
conda install python
pip install opencv-python

```
## Run:
```bash
python DinosaurGame.py
